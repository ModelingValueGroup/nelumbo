Each block below is a live Nelumbo editor. Press Run to evaluate it. A query line ends with ? and the result shows the facts and falsehoods that were inferred. Every block imports the stdlib module it needs, so it runs on its own.

Propositional logic

Import nelumbo.logic to work with truth values. Nelumbo evaluates each query and reports whether it holds. Variables of type Boolean get bound to the values that make the query true.

import nelumbo.logic true & true ? true | false ? !false ? Boolean p p & true ?

Integer arithmetic

Import nelumbo.integers for numbers. Queries run both ways: give the result and Nelumbo solves for the unknown. Below, a + 11 = 21 binds a to 10.

import nelumbo.integers Integer a 10 + 11 = a ? 10 * 11 = a ? a + 11 = 21 ? a > 0 ?

Rules and recursion

Declare a functor with ::= and give it a rule with <=>. This defines Fibonacci recursively: the base case for n <= 1 and the recursive case for n > 1. The query solves fib(7).

import nelumbo.integers Integer ::= fib(<Integer>) Integer n, f fib(n)=f <=> f=n if n<=1, f=fib(n-1)+fib(n-2) if n>1 Integer r fib(7)=r ?

Strings

Import nelumbo.strings for text. Concatenation with + is reversible, so "foo" + a = "foobar" binds a to "bar". Use len for length and str to convert a number.

import nelumbo.strings String a Integer d "foo" + "bar" = a ? "foo" + a = "foobar" ? len("foo") = d ? str(123456) = a ?
